SGB Dumper filer merger

This tool allows you to reconstruct the Super Game Boy system ROM image from files dumped by the Universal SGB Dumper tool, which copies the system ROM onto the Game Boy cartridge's save RAM. To download the ROM needed to extract the SGB system ROM, and details on how to use this tool, please see the documentation in the Universal SGB Dumper repo.

To use the tool, open or drag and drop partial files from the dumper. You can open the files in any order. The tool will automatically analyze the partial files and order the parts the parts correctly. Different revisions of SGB share data, so adding a file may register as completing parts of multiple ROM revisions. When all parts have been received, you can download the complete SGB system ROM image.

This tool runs completely on the client side. No information about the files you open is sent to a remote server.
